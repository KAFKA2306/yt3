# クリーンアップ (鉄の掟)

**目的**: 不要コード、コメント、無駄なエラーハンドリングの排除。

## 鉄の掟 (Rules)

1.  **削除対象 (Delete)**:
    -   未使用の関数、クラス、変数。
    -   **コメント** (コード自体で語れ)。
    -   `try-catch` ブロック (エラーは隠すな、落とせ)。
    -   リトライ/タイムアウト処理 (LangGraphの責務とせよ)。

2.  **維持対象 (Keep)**:
    -   実際に使用されているコード。
    -   型定義 (Zod/TypeScript)。

3.  **検証 (Verify)**:
    -   `npm run lint` または `npx tsc --noEmit` を実行し、エラーがないことを確認せよ。
