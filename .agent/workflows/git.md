# /git — Git操作ガイド

変更内容をリポジトリへ反映するための手順です。

## 手順

// turbo-all

1. `git status` で変更内容を確認してください。
2. `git add -A` で、すべての変更をステージング領域に追加します。
3. `git commit -m "feat: [変更内容の概要]"` で、コミットを作成します。
4. `git push` を実行し、リモートリポジトリへ反映します。
5. `gh run list` で、CI（自動テスト等）の実行状況を確認してください。
