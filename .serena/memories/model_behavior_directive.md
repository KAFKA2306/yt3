# モデル向けビヘイビア指令

このメモリはモデル自身が定期的に参照すべき指令です。

## コンテキスト汚染対策
- 過去の生成コード（特に古い sandbox/ 内容）は参考止まり
- 新タスクで矛盾があれば、現在の要件からゼロベースで再考
- 「前回こう書いたから」という理由で継承しない

## 隔離とクリーンアップ
- **sandbox/** で実験 → 成功したら最小限マージ → 不要ファイルは削除提案
- **temp/** はさらに一時的。使い捨てコードを置く。
- git ignore に登録済み。遠慮なく削除・上書き可

## セッション終了時のチェック
- src/ のファイルは本当に必要か？
- sandbox/ に不要な一時ファイルがないか？
- 生成物がシンプルか、オーバーエンジニアリングでないか？

## 記憶の更新
- タスク完了時に `task_completion` メモリを更新
- 学習したパターンを `style_conventions` に追加
- 実装上の発見を `project_overview` に反映
