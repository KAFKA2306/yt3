2026年3月14日現在の中東情勢（ホルムズ海峡の実質封鎖・原油供給中断）による日本個別株への影響を、セクター別・主な銘柄を中心にまとめます。情報は最新の市場報道・アナリスト見解に基づき、原油価格急騰（WTI一時119ドル超、現在80-90ドル台推移）と供給不安が主因です。全体の市場状況（参考）日経平均：3月上旬に急落（一時5万3000円台）、現在5万4000-55000円台で推移。
主な要因：原油高によるコスト増・インフレ圧力 → 企業収益圧迫、消費減速懸念。
影響は短期（リスクオフ） vs **長期（供給途絶）**で分かれるが、長期化すれば日本経済全体に深刻（GDP押し下げ0.3-0.65%試算）。

セクター別・主な個別株への影響1. エネルギー・石油関連（プラス影響大）原油価格上昇が直接利益増につながる「上流」セクターが逆行高。  INPEX (1605)：原油高の最大受益者。在庫評価益・権益価値向上で株価上昇（3月上旬+6%超の逆行高例多し）。PBR1倍近辺で割安感あり。
石油資源開発 (1662)：同様に上流中心で恩恵大。株価急伸傾向。
ENEOSホールディングス (5020) / 出光興産 (5019) / コスモエネルギーHD (5021)：下流（精製・販売）中心だが、在庫評価益で短期プラス。マージン悪化リスクはあるが、全面安相場で逆行高の場面あり。現在は神経質だが、備蓄在庫の再評価でポジティブ。
2. 海運（プラス影響大・ただし限定的）封鎖で船舶需給ひっ迫 → 運賃上昇期待。  商船三井 (9104) / 日本郵船 (9101) / 川崎汽船 (9107)：ホルムズ封鎖でタンカー待機・運賃高騰期待。3月上旬に+2%前後の逆行高。長期化ほど強いが、イラン側も輸出依存のため完全封鎖は現実的でないとの見方で上昇限定的。
3. 防衛・重工（プラス影響中）地政学リスク高まりで防衛予算増期待。  三菱重工業 (7011) / 川崎重工業 (7012) / IHI (7013)：防衛関連で注目。短期的に逆行高の例あり（3月12日時点で堅調）。ただ、全体リスクオフで一時調整も。
4. 化学・素材（マイナス影響大）原油高で原料コスト急増 → 減産・価格転嫁難。  三菱ケミカルグループなど：エチレン生産停止リスクで供給減。株価下落圧力強め。  
全体としてナフサ高で業績圧迫。長期化で深刻。
5. 電力・ガス（マイナス影響中〜大）燃料輸入コスト増 → 電力料金上昇・規制価格圧力。  東京電力HD / 関西電力など：LNG・原油高でコスト増。夏の電力不足リスクも。株価軟調。
6. 航空・輸送（マイナス影響大）燃料費急増 + 旅客減（中東経由影響）。  日本航空 (9201) / ANAHD (9202)：月100億円超の費用増試算。株価大幅安。
7. 自動車・消費関連（マイナス影響大）ガソリン高 → 消費減退・物流費増。  トヨタ自動車 (7203) / **ホンダ (7267)**など：大幅安。インフレによる購買力低下懸念。
8. ディフェンシブ・内需（相対的にプラス）リスク回避の逃避先。  オリエンタルランド (4661) / **東宝 (9602)**など：逆行高の例多し。  
小売の一部（イオンなど）はインフレで下押しも。
まとめと今後のポイント受益セクター：上流エネルギー（INPEXなど）、海運、防衛が短期的に強い。
被災セクター：化学、電力、航空、自動車がコスト増で苦戦。全体として日本株は輸入国ゆえの逆風が強く、日経平均は原油高と連動して下押し圧力。
現状の相場：原油高一服で一部反発も、封鎖長期化懸念でボラティリティ高。IEA備蓄放出で短期緩和期待はあるが、根本解決遠い。
長期化（3-6ヶ月）なら原油90-130ドル台持続で企業利益5%前後押し下げ試算。停戦見通しで反発期待。
